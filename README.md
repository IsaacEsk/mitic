# mitic

Simple Flutter game project scaffolded for web (desktop/mobile), iOS, and Android. An example `main.dart` displays a basic card.

## Getting Started

This project is a starting point for a Flutter application targeted at multiple platforms. Initially, development and testing will be done on web.

### Running the App

```bash
flutter run -d chrome
```

### Next Steps
- Replace card content with game logic.
- Add assets and additional pages as needed.
- Configure platform-specific settings for iOS/Android when ready.

For general Flutter guidance, see the [online documentation](https://docs.flutter.dev/).
